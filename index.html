Visual Studio Code

Visual Studio Code is a code editing platform made by Microsoft for macOS, Windows, and Linux.

You need to set it up by downloding VS code on Microsoft, macOS, or Linux; crete a new folder, file; install extensions for your programming language; and chose your theme.

Console

The console.log() function is used to print variables in the console screen. The console screen is also used to send a message to the user.

To open the console screen you have to right click, then click on insert, and then press the console button.
